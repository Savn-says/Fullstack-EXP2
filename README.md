# Experiment-1: Designing UI Using Bootstrap Components

## Aim:
The aim of this experiment is to design a responsive user interface using **Bootstrap** components in a **React** application. This will help us understand how to leverage Bootstrap's pre-built UI components to create a user-friendly, mobile-responsive design with minimal custom CSS.

## Software Requirements:
- **Node.js**: A JavaScript runtime environment needed for running React applications and installing packages via npm.
- **React**: A JavaScript library for building user interfaces, particularly for creating Single Page Applications (SPA).
- **Bootstrap**: A CSS framework providing a set of pre-designed components like navigation bars, forms, buttons, and cards that facilitate the creation of responsive and consistent UIs.
- **Text Editor**: A text editor like **VS Code** or **Sublime Text** to write and manage the code.
- **Web Browser**: To test and view the application.

## Theory:
Bootstrap is a popular front-end framework that allows developers to create clean, professional-looking, and responsive websites without writing a lot of custom CSS. It includes components like:
- **Navbar**: A responsive navigation bar used for website headers.
- **Cards**: Used to display content in neat, organized blocks.
- **Forms**: Essential for gathering user input like login and registration forms.
- **Buttons**: Pre-styled interactive elements that trigger user actions.
- **Alerts**: Used to show messages or notifications to users.

## Components Used in the Experiment:
1. **Navbar**: This is used for navigation and is typically displayed at the top of the page. It can contain links to different sections of the website.
2. **Card**: A Bootstrap component that allows you to display content such as text, images, or buttons inside a styled container.
3. **Form**: Forms are used for collecting user input, such as email and password fields for login or registration.
4. **Button**: Buttons are interactive elements that perform an action when clicked, like submitting a form or opening a new section.
5. **Alert**: Alerts display information or status messages to the user, such as success, failure, or informational messages.

## Procedure:
1. **Setup React App**: Begin by setting up a new React project using `create-react-app` if you haven't done so already.
2. **Install Bootstrap**: Use npm to install Bootstrap into your project to access its pre-built components.
3. **Import Bootstrap CSS**: Import the Bootstrap CSS file into the `index.js` file of your React application to apply the Bootstrap styles globally.
4. **Design the UI**: Utilize Bootstrap components such as the navbar, cards, forms, buttons, and alerts to design the interface of your application.
5. **Run the Application**: After designing the UI, run the React application using `npm start` to launch it in the browser. You can now interact with the application and see how the Bootstrap components work.

## Expected Results:
The application should display a responsive user interface consisting of:
- A top navigation bar for accessing different sections.
- Cards that organize content into visually appealing blocks.
- A form for user input (e.g., login form).
- Buttons that trigger actions like submitting the form.
- Alerts that notify the user of important information.

## Conclusion:
This experiment demonstrates how to quickly build a responsive and attractive user interface using **Bootstrap components** within a **React application**. By utilizing these pre-designed components, we can significantly reduce the amount of time and effort required to create a functional and visually appealing UI.

# Exp 2-Mountain Cards Web Project

## Description

This project displays a webpage featuring cards with information about three famous mountains: **Mount Everest**, **K2**, and **Kangchenjunga**. Each card includes an image, a brief description, and a link to the mountain’s Wikipedia page.

- **Dynamic Cards**: Cards are generated using JavaScript.

## Technologies

- **HTML**
- **CSS**
- **JavaScript**
- **Bootstrap**

## How to Use

1. **Download/Clone** the project.

## Overview

A simple React app with a responsive navbar and dynamic section colors. The background color of each section and the navbar color change based on the active route.

- **Routing**: Uses React Router for navigation.

## Technologies Used

- **React**
- **React Router**
- **React Bootstrap**
- **CSS**

## Setup

1. **Install Dependencies**:

   npm install react-bootstrap bootstrap react-router-dom

2. **Run the App**:
